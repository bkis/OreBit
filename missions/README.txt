MISSION CREATION NOTES:

1) mission files have to be named like "mission_0.xml"
   (insert order number, start from 0, no gaps!!!)

2) you can copy and modify "template.xml" to create missions!
   the template contains every possible element a mission can have!

3) put all the mission files in a zip-archive and place it in the "missions" folder!

4) main campaign archive has to be named "campaign.zip"
